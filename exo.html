Qu'est ce que HTML?

  • C’est le langage de balisages tandard pour la création de pages Web.
  • Il décrit la structure des pages Web en utilisant le balisage    
  • Il permet de formater du texte, ajouter des graphiques,
    créer des liens, des formulaires, des blocs, etc.,
    et enregistrer le tout dans un fichier texte
    qu’un navigateur peut lire et afficher.

Structure d’un document HTML

ExempleStructure
<!DOCTYPE html > La déclaration <!DOCTYPE html > définit que ce document est du HTML5
< html > L'élément < html > est l'élément racine de la page
< head > L'élément < head > contient des méta informations sur le document
<title> Titre de la page </title> L'élément < title > spécifie le titre du document
< /head >
< body > L'élément < body > contient le contenu visible de la page
< h1 > Ma première entête < /h1 > L'élément < h1 > définit un groupe titre
< p > Mon premier paragraphe < /p > L'élément < p > définit un paragraphe
< /body >
< /html >

Document HTML = Page Web

          Un document HTML est un fichier d’extension .htm ou .html décrivant une page web
	Un document HTML est aussi appelé page web
	Il contient des balises HTML et du texte brut
	Le but d'un navigateur web (comme Internet Explorer ou Firefox) est de lire les documents HTML et de les afficher sous forme de pages Web
	Le navigateur n'affiche pas les balises HTML mais les utilise pour interpréter le contenu de la page

Les balises HTML

  1. Les balises HTML sont des noms d'éléments entourés de crochets
  2. Syntaxe : <nomelement> le contenu </nomelement>
  3. Les balises HTML sont normalement par paire comme <b> et </b>
  4. La premiere balise dans une paire est la balise de debut, la seconde balise est la balise de fin
  5. La balise de fin est écrite comme la balise de début, mais avec une barre oblique insérée avant le nom de la balise.</html>

Editeurs HTML

Les pages Web peuvent être créées et modifiées
à l'aide d'éditeurs HTML professionnels comme Dreamweaver et Visual Studio.
   

Cependant, pour apprendre le langage HTML, nous recommandons
un simple éditeur de texte comme Notepad++ ou TextEdit (Mac).
   

Les titres en HTML

Les titres sont définies avec les balises <h1> à <h6>. <h1> définit le titre le plus important et <h6> le moins important.

Exemple

<h1>Ceci est un titre 1</h1>    Ceci est un titre 1

<h2>Ceci est un titre 2</h2>    Ceci est un titre 2

<h3>Ceci est un titre 3</h3>    Ceci est un titre 3

<h4>Ceci est un titre 4</h4>    Ceci est un titre 4

<h5>Ceci est un titre 5</h5>    Ceci est un titre 5

<h6>Ceci est un titre 6</h6>    Ceci est un titre 6

Les commentaires en HTML

Les commentaires peuvent être insérés dans le code HTML pour le rendre plus lisible et compréhensible. Les commentaires sont ignorés par le navigateur et ne sont pas affichés.

Syntaxe :    <!-- Ceci est un commentaire -->
